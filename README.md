# SynthDataGen

A comprehensive tool for generating synthetic tabular data using various state-of-the-art methods, CART (Classification and Regression Trees) and SDV (Synthetic Data Vault) models.

## Installation

1. Clone the repository.
2. Install the required dependencies:

```bash
pip install -r requirements.txt
```

## Usage

1. Place your dataset (CSV or Excel) in the `datasets/` folder.
2. Run the main script:

```bash
python run_all.py
```

3. Output files are:
   - Synthetic datasets in `datasets/`
   - PCA Plots in `plots/`
   - `evaluation_report.xlsx`.
